# similar-flowers
Поиск похожих картинок цветов. Итоговый проект в Иннополис.

## Как запустить
```bash
cd backend
pip install -r requirements.txt
python main.py
```

## Какие технологии использовались
<ol>
  <li>PyTorch</li>
  <li>ONNX</li>
  <li>OpenCV</li>
  <li>albumentations</li>
  <li>FAISS</li>
  <li>FastAPI</li>
  <li>React</li>
</ol>
