# similar-flowers
Поиск похожих картинок цветов. Итоговый проект в Иннополис.

## Как запустить
Запустить сервер
```bash
cd backend
pip install -r requirements.txt
python main.py
```
Перейти по ссылке на веб-морду - <a hred="http://127.0.0.1:8000/">http://127.0.0.1:8000/</a>
